# Event Management System

A full-stack Event Management System built using Java Spring Boot, React, MySQL, and JWT Authentication.

## Tech Stack

### Backend
- Java 21
- Spring Boot
- Spring Security
- JWT Authentication
- Spring Data JPA
- Hibernate
- MySQL

### Frontend
- React.js
- Axios
- Bootstrap

## Features

- User Registration
- User Login
- JWT Authentication
- Create Event
- View Events
- Update Event
- Delete Event
- Secure REST APIs

## Project Structure

```
EventManagementSystem
│
├── backend
│   └── eventmanagement
│
├── frontend
│
├── database
│
└── docs
```

### Authentication

### Events
